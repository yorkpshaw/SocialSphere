```
npm init -y
npm i express body-parser bcrypt cors dotenv gridfs-stream multer-gridfs-storage helmet morgan jsonwebtoken mongoose
```
```
body-parser: parses the request body
bcrypt: password encryption
cors: cross-origin resource sharing
dotenv: environment variables
gridfsstream: file upload
multer-gridfs-storage: upload files locally
helmet: request safety
morgan: logging
jsonwebtoken: authentication
mongoose: MongoDB access
```
